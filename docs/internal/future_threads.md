# Future Threads

## Purpose

This document parks future lines of work that are clearly promising but should not drive the first `aimez` reorganization pass. These are not low-value ideas. They are simply not phase-1 priorities.

## Thread 1. Polymers / condensates / active matter as a bridge to the synapsin simulations

### Why it matters

The user flagged a possible bridge between:

- polymers and condensates
- allocentric flocking and collective behavior
- synapsin simulations already run elsewhere

This is potentially powerful because it could connect:

1. the active-matter / collective-behavior line (boids, crowds, allocentric flocking)
2. the structural / allosteric / rigidity line
3. the biological simulation work around synapsin

If that bridge is real, it would make `aimez`'s cross-domain posture substantially stronger because it would show that the same structural commitments extend from urban routing and HAII all the way into biological materials and molecular organization.

### Why it is not phase-1 priority

Because it is still one level too unconstrained. Right now the bridge is a promising direction, not yet a packet, not yet a clean piece of the public-facing identity.

To force it into the phase-1 strategy pack would create noise in exactly the place the first phase is trying to remove.

### What should happen later

When this thread is opened, the first task should be to identify the smallest coherent comparative object. Options:

1. one short internal memo connecting condensate-like phase behavior to active-matter / collective-motion framing
2. one private theory note mapping synapsin simulation outputs to the same first-order / second-order / field-level language used in the mobility substrate
3. one literature map showing what, if anything, is shared between:
   - phase transitions in materials
   - allocentric / egocentric navigation and flocking
   - substrate-level allostery in the mobility work
   - synapsin simulation behavior

Do not open with a full public-facing synthesis. Start with a controlled internal memo.

## Thread 3. Formal non-smooth analysis beyond the current Sane anchor

### Why it matters

The regularity-gap idea in Sane et al. points toward a deeper mathematical thread:

- piecewise-smooth systems
- Clarke generalized Jacobians
- subdifferential calculus
- hybrid dynamical systems

The mobility substrate and the HAII structural argument both rely on non-smooth second-order fields (shed activations, threshold crossings, abrupt regime shifts). This suggests a richer theory connection than the current public site should carry.

### Why it is not phase-1 priority

The immediate need is posture and architecture, not a formal math note. The user has already identified the regularity-gap idea as strong enough for Q&A and Jen-facing materials; that is sufficient for now.

### What should happen later

Possible future document:

- `docs/internal/non_smooth_structure_note.md`

It would ask:

- how much of the `aimez` problem is really a non-smooth analysis problem?
- when do thresholds and discrete events need formal treatment?
- where can the current language stay qualitative?
