O0 SMS — Recipient consent & message flow

Campaign collateral for A2P 10DLC. Last updated: 2026-07-21.

Who receives messages

Only one enrolled operator handset. This is a private operational alert channel (hard-stop notices). Not marketing. Not a public list.

How end-users opt in

  1. The account owner reads the Terms and Privacy Policy (links below).
  2. The owner enrolls their personal mobile number offline in a private config used only by the O0 SMS tool (no public web form, no purchased lists).
  3. Optional text confirmation: the enrolled user may text START to the program long code +1 (360) 317-2476.
  4. They receive a confirmation / welcome message (sample below).
