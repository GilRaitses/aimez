Corridor topology primer

This page explains how the Midtown pedestrian-graph work can be read with topology vocabulary (paths, loops, residuals) without claiming a finished homological theorem. It is a modeling frame for the measured stress field and route behavior on a fixed corridor graph.

Interactive routing demo · Graph-field research overview

The modeled system

Manhattan Midtown is represented as a pedestrian graph. Camera-derived stress and public capacity layers assign values to directed edges. Routing compares distance-only paths with paths that read the measured field.

Paths and loops

A route is a path-like trace. A closed walk tests whether local edge costs cancel around a loop or leave a residual. The project uses loops as probes, not as a claim about the fundamental group of the city.

Chains, cycles, and residuals

Directed edges can be treated as oriented generators. A route is an ordered sum of edges. A residual is what remains when costs do not cancel around a loop. Basis checks ask whether patterns persist under an equivalent loop organization.

Topology language here formulates tests; it does not replace empirical validation or claim invariants for Manhattan as a whole.

Scope limits

  • No homology groups computed for the corridor graph in this public note.
  • No fundamental-group result claimed.
  • Loop-closure and basis-stability results remain work in progress.
