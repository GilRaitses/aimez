    How it works

    Public traffic-camera frames become a per-block stress field on the canonical Midtown-corridor graph. The router runs A* with edge cost 1 + alpha * stress (alpha = 1.0 in this demo) so calmer blocks are cheaper. Each leg returns a shortest (block-minimizing) path and a stress-aware path, plus shade-aware, comfort (LST + shade), and adaptive-comfort variants — all on the same corridor graph. The comparison figure shows these routes side by side with a comfort phase map; the metrics table below summarizes the shortest vs stress-aware comparison.
